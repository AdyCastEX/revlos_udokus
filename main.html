Background

Sudoku is a logic-based, combinatorial, number-placement puzzle, where the objective is to fill out an n×n grid (usually 9×9) with digits such that each row, column, and subgrid (each grid has n subgrids, with dimensions √n×√n) contain all the digits from 1 to n. Commonly, n is a perfect square so that √n resolves to a whole number. For example, the common 9×9 grid has 9 3×3 subgrids.
